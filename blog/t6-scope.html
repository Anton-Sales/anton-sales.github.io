JavaScript Scopes

What is a scope and how does it work in JavaScript?

There are two types of scopes, 'Global Scope' and 'Local Scope'. Global scope is a variable declared outside of a function and can be used in all scripts. A local scope is a variable that is limited to one function and that function only.
