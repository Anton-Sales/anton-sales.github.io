JavaScript Basics

How does JavaScript compare to HTML and CSS?

The main difference with JavaScript with HTML and CSS is that JavaScript is used to make webpages interactive. HTML is responsible for the content of the site and the basic structure. CSS upgrades this basic structure by adding style. JavaScript gives the webpages functionality and adds that dynamic effect to bring webpages to life.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is the order of which the computer executes statements in a script. This is done by the order of the code is run from the first line to the last line. Loops can be quite tricky, A basic example of this is a 'for' loop, which is declared with three optional expressions separated by semicolons. ([initialization]; [condition]; [final-expression]). A basic example would look like this - var a = time; if (a < 7) { snooze; } else { Wake Up; }

Explain the difference between accessing data from arrays and objects.

The main difference accessing data from array and objects is that array uses numbered indexes and objects use named indexes. Arrays could be accessed using brackets and the index number and objects could be accessed by ‘objectName.propertyName’.

Explain what functions are and why they are useful.

Functions is a block of code, which is a procedure that performs tasks' or calculates a value. This is useful for us simply because the code is reusable. Once the code is defined, you will be able to use it in the future with different arguments.
